## ðŸ“‹ Empty State Library (Templates)

### 1. No Search Results
*   **Icon**: Magnifying glass with "X".
*   **Headline**: "No results for '[Query]'"
*   **Explanation**: "Try a different search term or clear filters."
*   **CTA**: [Clear Filters] button.

### 2. No Past Items (History/Orders)
*   **Icon**: Clock or Receipt.
*   **Headline**: "Your history is empty"
*   **Explanation**: "Your past activities will show up here."
*   **CTA**: [Explore Services] / [Browse Products].

### 3. No Favorites/Saved
*   **Icon**: Heart (Outlined).
*   **Headline**: "No favorites yet"
*   **Explanation**: "Tap the heart icon on any item to save it here."
*   **CTA**: [Browse Now].

### 4. Connection Error (Temporary)
*   **Icon**: Cloud with "X".
*   **Headline**: "Couldn't load content"
*   **Explanation**: "Check your connection and try again."
*   **CTA**: [Retry] button.
