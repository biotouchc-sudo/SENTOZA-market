# Journey Architecture

**Rule:** Build Journeys, not Pages.

## Defined Journeys

### 1. Discovery Journey (The "Hook")
**Goal:** Move user from "Unaware" to "Interested".
**Required Steps:**
1. **Hero:** Problem/Solution statement (3s rule).
2. **Social Proof:** Why trust us?
3. **Value Prop:** What do I get?
4. **Exit:** CTA to Decision Journey.

### 2. Decision Journey (The "Commitment")
**Goal:** Remove friction and objections.
**Required Steps:**
1. **Pricing/Selection:** Clear options.
2. **Objection Handling:** FAQ/Guarantee near CTA.
3. **Risk Reversal:** "Cancel anytime" / "Free trial".

**Goal:** Deliver the promised outcome instantly.

2. **Success State:** Celebration of first win.
3. **Retention:** Prompt for next step.
