# SENATE NON-NEGOTIABLES

1. **No Design Without Intent:** You cannot build a component without stating its persuasive goal.
2. **No Code Without Budget:** You cannot merge without passing Performance Budget.
3. **No Voice Mixing:** You cannot mix "Corporate" and "Friendly" tones.
4. **No Copy-Paste:** You must generate 3 variations before selecting one.
5. **No Dead Ends:** Every screen must have a primary action.
6. **No Silent Failures:** Every error must teach the user.
7. **Accessibility is Default:** Not an "enhancement".

**Violation of these rules is a FAILURE of the Engineering Role.**
