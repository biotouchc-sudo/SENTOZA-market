# Landing Page (High Conversion) - UX Preset

## 1. Interaction Physics
- **Scroll Damping**: 0.85 (Heavy, luxurious feel)
- **Hover Delay**: 0ms (Instant feedback)
- **Click Latency**: <50ms

## 2. Animation Tiers
### Tier 1: Hero Section
- **Entrance**: Staggered fade-up (0.1s delay between elements)
- **Background**: Subtle parallax or slow pan (never distracting)
- **Button**: Magnetic effect (attracts cursor within 20px)

### Tier 2: Features Grid
- **Scroll Trigger**: Reveal when 20% in view
- **Hover State**: 1.02x Scale + Glow (Amber-500/20)

### Tier 3: Trust Signals
- **Infinite Marquee**: Slow continuous scroll (30s duration)
- **Testimonials**: Auto-play disabled (user control only)

## 3. Cognitive Load
- **Primary CTA**: Only one per viewport height (Screen)
- **Text Density**: < 60 characters per line
- **Contrast Ratio**: AAA for body text, AA for headings

## 4. Mobile Adaptation
- **Touch Targets**: Min 48x48px
- **Sticky Elements**: Only "Book Now" bar at bottom
- **Animations**: Reduced motion (fade only)

> [!TIP]
> Use this preset for: Home Page, VIP Landing Page, Special Offers.
